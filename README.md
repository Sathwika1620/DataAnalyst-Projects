# Data Analyst Projects Portfolio

This repository contains end-to-end data analysis and visualization projects
built using Excel, Power BI, Tableau, and Python. The projects focus on
analyzing business data and presenting actionable insights.


## Projects

### 1. Production & Sales Performance Dashboard (Excel)
- Analyzed production, revenue, profit, expenses, and transactions data.
- Created charts for profit by payment method, revenue by product line,
  category-wise performance, regional transactions, and departmental expenses.
- Used pivot tables and charts to build an interactive Excel dashboard.

### 2. Insurance Data Analysis Dashboard (Power BI)
- Built an interactive Power BI dashboard using insurance datasets.
- Analyzed claims, policy types, and customer risk patterns using DAX measures.
- Visualized KPIs to support business and operational insights.

### 3. Superstore Sales & Profit Analysis Dashboard (Tableau)
- Created an interactive Tableau dashboard using the Sample Superstore dataset.
- Analyzed sales, profit, and order quantity across regions and customer segments.
- Visualized profit vs sales, regional sales distribution, and monthly order trends.

### 4. Profit Analysis Using Python
- Performed exploratory data analysis using Python, Pandas, and Matplotlib.
- Analyzed profit trends and segment-wise performance.
- Generated visual insights to support business decision-making.
